# MyBeerBuletin

The app which allow you to find your favorite Beer and get the details of the Brewer and Brewery. You can also read reviews and post your experience.

<img src="./client/assets/beerBuletin.jpg" />

## Tech Stack

- React Native
- MongoDB
- Mongoose
- Express
- Jest
